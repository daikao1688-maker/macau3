## 🚀 Quick Start

### Requirements
- Node.js 22.12.0 or higher
- npm, yarn, or pnpm

### Installation

```bash
# 1. Extract ZIP file
unzip CasinoPro-Premium-Theme-v1.0.0.zip
cd CasinoPro-Premium-Theme

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev

# 4. Open browser
# Visit: http://localhost:4321
```

### Build for Production

```bash
npm run build
```

Output will be in `dist/` folder.

## 🎨 Customization

### Easy to Customize

**Colors & Branding:**
- Edit `tailwind.config.mjs` for colors
- Replace logo in `public/` folder
- Update meta tags in `src/layouts/Layout.astro`

**Content:**
- Use TinaCMS at `/admin/index.html` for home, promotions, games, and providers
- Edit CMS source files in `content/casino/`
- Modify pages in `src/pages/`
- Update components in `src/components/`

**Demo Games:**
- Customize themes in `src/data/demoThemes.ts`
- Modify demo components in `src/components/`
